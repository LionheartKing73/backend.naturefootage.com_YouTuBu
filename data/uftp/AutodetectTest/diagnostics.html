FileCatalyst Diagnostics

Overview

Requirements

Java 1.6 or higher; a FileCatalyst Server (trial or production).

How it works

A Java applet loads in the background, enabling a true client-side connection; you are not simply testing the website hosting this page. This applet connects to a FileCatalyst Server (it will not test any other type of server), runs a suite of tests, and displays results in a table.
